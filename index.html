Terabox Proxy Server

This is a proxy server for handling Terabox downloads.

API Usage

To use this proxy, make a GET request to:

/api/proxy?url=https://terabox.com/your_file_link

Example:

https://your-vercel-app.vercel.app/api/proxy?url=https://terabox.com/s/1abCDef2GhIjKl_123456

Notes

  • Only Terabox URLs are allowed
  • Maximum response time: 60 seconds
  • Request will be forwarded with appropriate headers
